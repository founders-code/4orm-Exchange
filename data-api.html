4ormEx is a design preview of a planned permissioned trading venue (Pillar 02 of six) inside the future 4orm Finance regulated control plane. It is not a registered marketplace, dealer, exchange, or bank in any Canadian jurisdiction; no applications have been filed. Every figure, identifier, counterparty and listing on this page is simulated and illustrative. Operating a marketplace for securities is a registrable activity in Canada under NI 21-101, 23-101, 24-102, and 31-103. Questions: compliance@4ormfinance.com.

Authentication

Every request is authenticated with a Bearer API key issued from the institutional console. Keys are scoped to a tier (sandbox or production) and an entity. All production-tier calls additionally require a signed Corporate NFT lookup so the API can resolve the calling entity's good-standing.
